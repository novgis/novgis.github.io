GeoServer 用户手册[译]

Geoserver 是一款用 Java 编写的开源地图服务器，允许用户共享和编辑地理空间数据。利用 GeoServer 可以很方便的发布基于开放标准的地理空间数据。

这份用户手册从各个方面综合性地介绍了 GeoServer，无论你是新手还是老司机，都希望这份文档对你有所帮助。

本系列文章是我对学习 GeoServer (v2.7.6) 过程的总结，如有错误，欢迎指正！
