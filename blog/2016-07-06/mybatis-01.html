(1) MyBatis 快速入门

  • mybatis
programming

MyBatis 是支持普通 SQL 查询、存储过程和高级映射的优秀持久层框架。MyBatis 消除了几乎所有的 JDBC 代码和参数的手工设置以及对结果集的检索封装。MyBatis 可以使用简单的 XML 或注解用于配置和原始映射，将接口和 Java 的 POJO 映射成数据库中的记录。

首先回顾下常见的一些持久层框架：
→ JDBC：Connection Statement PrepareStatement ResultSet
→ DBUtils：自动封装查询结果集，但是要在代码中写 SQL 语句且传参数需要一个个传
→ Hibernate：不需要写任何 SQL 语句，自动生成那就可能存在浪费
→ MyBatis：相比 DBUtils 它不用在代码中写 SQL 语句，而是在配置文件中写；相比 Hibernate 的自动生成，其手动写 SQL 增加了一些灵活性
