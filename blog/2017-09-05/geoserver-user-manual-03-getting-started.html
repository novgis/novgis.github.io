Web管理界面简介

GeoServer 通过 Web 管理界面管理所有配置项，包括新增、发布地理空间数据以及服务的一些配置等。

访问 Web 管理界面的地址格式 http://<host>:<port>/geoserver，本机安装默认地址为 http://localhost:8080/geoserver

启动 GeoServer 服务之后，访问如上链接，浏览器将显示 GeoServer 的欢迎页：

管理员登录

用户必须先通过授权登录之后，才能管理服务的配置和数据。

在欢迎页右上角输入用户认证信息登录管理器，本机安装默认用户名 admin，默认密码 geoserver

注意：详细了解用户配置请参阅 十三、安全策略 章节介绍

管理员登录后，左侧目录树将展示所有的管理功能：

图层预览

  1. 点击左侧目录树中 Layer Preview。

  2. 在图层预览管理页中加载了当前 GeoServer 服务器上所有的图层信息，并提供在线预览和导出为其它格式的功能。
  3. 点击列表表头可按字母顺序排序，方便查找。
  4. 另外在图层太多的时候，可以通过列表右上角的搜索框输入查询条件，按回车键后列表将展示您的搜索结果

注意：详细了解图层预览请参阅 五、数据管理 章节介绍
