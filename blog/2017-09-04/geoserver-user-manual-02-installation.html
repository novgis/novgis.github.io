二、GeoServer 安装部署[译]

  • geoserver
gis

有很多方式在各操作系统中安装 GeoServer，本节就来讨论这些方式。

如果您用的是 Windows 或 OS X 系统，我们建议您使用 installers 进行安装。

Windows安装程序

Windows安装包提供了最方便的方式来安装 GeoServer，无需各种配置文件和命令行设置。

  1. 确定系统已正确安装 Java Runtime Environment(JRE)。GeoServer(v2.7.6) 安装的先决条件是 Java 7 运行时，最好是 Oracle JRE，OpenJDK 也能勉强使用。您可以从这里下载 Oracle JRE 7。

    注意： GeoServer v2.7.6 暂时还不支持 Java 8， 更多有关 GeoServer 和 JRE 版本支持请参考 Java Consideration

  2. 打开下载页面

  3. 选择您要下载的版本号，如果没有特别需求，请选择稳定版
  4. 点击 Windows Installer 链接下载 GeoServer Windows 安装包
  5. 下载结束运行 GeoServer 安装程序

    此处省略一万步… 很简单，一路 Next！

  6. 安装完成后，如果 GeoServer 被部署为 Windows 服务，则已经启动；否则在开始菜单中找到 GeoServer 目录，点击 Start GeoServer 即可启动
  7. 启动成功后，在浏览器中输入 http://localhost:8080/geoserver 即可访问 GeoServer 的 Web 管理器界面
  8. 卸载 执行 GeoServer 安装目录中的 uninstall.exe，或者通过系统的卸载程序卸载

版本更新

警告： GeoServer 版本升级有可能是不可逆的，版本升级后 data directory 有可能就不再支持旧版本了，所以在进行版本升级前请仔细阅读 版本迁移需知

正常的 GeoServer 版本迁移处理流程如下：

  1. 备份现有的 data directory，比如拷贝一份到其他目录中
  2. 确保 data directory 不在 GeoServer 的安装目录下
  3. 卸载旧版本安装新版本，也可以直接覆盖
  4. 将新版本的 data directory 重新指定为之前备份的目录

注意一些特殊版本的更新：

从 GeoServer 2.6 开始，WFS 服务产生的 GeoJSON 将不再使用非标准的坐标参考系(CRS)，为了向后兼容需要将 GEOSERVER_GEOJSON_LEGACY_CRS=true 参数设置为系统属性、上下文参数或环境变量
